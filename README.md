# HTTPBench
## 开发语言
C/C++
## 开发环境
CentOS8、vscode、g++、git、MakeFile
## 项目介绍
HTTP是一个基于多线程实现的HTTP压力测试工具

支持版本：HTTP1.1
请求类型：GET
支持参数：

-n(总连接数，默认为1000000)， 

-c(并发数，默认为1)， 

-t(测试时间， 默认为30s),  

-p(使用代理服务器，

-r(不进行缓存）
